# Snake Game

Змейка на Java + JavaFX. Поле 20×20, цель - вырасти до длины 70.

## Управление

**Клавиатура:** `W A S D` или стрелки.

**Arduino:** при запуске игра ищет COM-порты и предлагает выбрать способ управления. Arduino шлёт байты `U` / `D` / `L` / `R` по Serial.

## Еда

На поле всегда 15 единиц еды, после поедания сразу спавнится новая. Тип выбирается случайно с весами.

| Тип | Эффект | Вес |
|-----|--------|-----|
| Яблоко | +1 сегмент | 70 |
| Бустер | скорость x1.33 | 15 |
| Перец | -1 сегмент | 15 |
| Лёд | скорость x0.67 | 15 |

Скорость зажата в диапазон **0.5x - 3.0x**.

## Код

**`Game`** - модель. Хранит змейку и карту еды, на каждом тике двигает змейку, проверяет столкновения и победу. Команды направления буферизуются в очередь на 2 нажатия, разворот на 180° игнорируется.

**`Snake`** - тело в виде двусвязного списка. `grow()` откладывает удаление хвоста до следующего хода, `shrink()` убирает хвост сразу.

**`Food`** - интерфейс с методом `applyEffect(Game)`. Каждый тип еды - отдельный класс с аннотацией `@SpawnWeight`.

**`FoodRegistry`** - находит все реализации `Food` через рефлексию и делает взвешенный выбор. Новый тип еды - просто новый класс с аннотацией и текстура в ресурсах, больше ничего.

**`GameController`** - игровой цикл через `AnimationTimer`, рендер спрайтов с поворотом под направление.

**`Main`** - грузит FXML, ищет COM-порты, настраивает ввод.
